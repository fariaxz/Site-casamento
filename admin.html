Sobre e-mails

ℹ️ Como funcionam os e-mails nesta versão:
Neste arquivo HTML, os e-mails são simulados — nenhum e-mail real é enviado. Para e-mails funcionarem de verdade, você precisa do projeto Next.js configurado com:

• Gmail: ative verificação em 2 etapas → gere uma "Senha de App" → configure no .env.local
• Resend.com (recomendado — mais fácil): crie conta gratuita, pegue a API Key e configure

No projeto Next.js, cada confirmação de presença e compra de presente dispara e-mail automático para noivos123@gmail.com.
